The main game features, instructions how to download and play, already somehow accomplished parts and plans

This is a highly social game in the phantasy setting of civilization progression.
Almost all content should be creatad by players, like in EVE, but in a much greater extent.
We will give players a wild interesting world, but without towns, NPCs, quests and so on.
It will be players who will build the towns, publish requests (quests), start wars, maintain the economy...
In order to make it happen, we need a huge server (or a couple of them), where numerous people will create their civilizations in their planet.. or, maybe, in their universe?

I used to think a lot about different aspects of the game. Now, as I started programming, I made some notes in this Unity project, but unfortunately they are in russian for now.
You can check it out simply by downloading the whole repository attached below if you have Unity - just open the project 'tutorial1' and enter the playing mode as shown in this gif:
