All my game projects pursue the goal to get closer to my ideal game. This is my first attempt of doing this, the first Java project, and the first code project at all.

The idea was to make a decent walking simulator with rpg elements, but eventually I realized, that it will take a lot of time with plain Java, and the work process stopped at the condition of moving sprites.

Though, the project has several feaures, written fro the Java scratch: simple movement - walk and run, ejections of the map elements from each other (depending on whether the element is static or not), inventory, and the most observable part of the project - A* pathfinding algorithm with a possibility to place obstacles.
